=== AIDE POUR LES CONTROLES ===



Menu de sélection des niveaux :

[GAUCHE, DROITE] Changer de niveau
[ENTREE] Charger le niveau


========================================================================


En jeu:

Faites arriver chaque personnage sur une case rouge puis activez la en case verte

[GAUCHE, DROITE] Déplacer le point de vue de la caméra
[ESPACE] Zoomer / Dézoomer
[ECHAP] Quitter

[Bouton vert] Lancer la séquence
[Bouton jaune] Mettre la séquence en pause
[Bouton bleu] Exécuter l'action suivante, puis mettre en pause
[Bouton rouge] Réinitialiser le niveau (les séquences restent intactes)

Actions dans l'ordre (dans le menu Action) :

Avancer,
Tourner de 90 degrés à gauche,
Tourner de 90 degré à droite,
Ne rien faire, 
Tourner la caméra de 90 degrés à gauche, (dispoible uniquement pour le robot principal)
Tourner la caméra de 90 degrés à droite (dispoible uniquement pour le robot principal)


========================================================================


Dans l'éditeur :

[Z,Q,S,D,HAUT,BAS] Déplacer le curseur
[ESPACE] Zoomer / Dézoomer
[R] Afficher / Masquer les aides spatiales
[A,E] Changer de bloc
[ENTREE] Poser un bloc
[DELETE] Supprimer un bloc
[ECHAP] Quitter

[O] (uniquement sur un bloc) passe en mode Linkage, avec le bloc sélectionné comme ancre

En mode linkage :

[ENTREE] Linker / délinker un bloc à l'ancre
[P] Affiche les paramètres de l'ancre (direction, nombre de cases)
[O] Pour sortir du mode linkage
